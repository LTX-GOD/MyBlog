hugo搭建

mac

  1. 安装hugo

brew install hugo

  1. 创建博客

    hugo new site myblog

  2. 创建主题

    cd site_name git init git submodule add https://github.com/dillonzq/LoveIt.git themes/LoveIt (git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt 也可以) echo theme = “LoveIt” » hugo.toml

当然你也可以选择其他的主题，方法与之类似

  1. 修改配置 你知道的，hugo.toml里面就是你整个博客的配置，包括博客名称，作者名字，头像，联系方式等，直接上模板
